# MTG Resolution Timeline

During MTG Commander games, capture card rules text (OCR) and organize reminders into turn-step buckets so you can resolve triggers/effects in order.

- iOS / Android only
- Session-only: Reset wipes everything

## Docs
- Product + UX: `docs/product/` and `docs/design/`
- Storage/OCR constraints: `docs/tech/`
- Roadmap + checklists: `docs/dev/`

## Dev
- Deps: `flutter pub get`
- Run iOS: `flutter run -d ios`
- Run Android: `flutter run -d android`
- Tests: `flutter test`
